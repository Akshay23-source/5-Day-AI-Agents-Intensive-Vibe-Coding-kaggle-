# 🤖 5-Day AI Agents: Intensive Vibe Coding Course With Google (Kaggle)

> **Completed:** June 15–19, 2026
> **Platform:** Kaggle × Google
> **Duration:** 5 Days (Intensive Course)

## 📖 Overview

This repository documents my learning journey through the **5-Day AI Agents: Intensive Vibe Coding Course With Google**, hosted by Kaggle and Google.

The course focused on designing, building, evaluating, securing, and deploying modern AI agents using Google's latest AI technologies. Throughout five intensive days, I explored how autonomous AI agents are built, how they interact with tools and APIs, how they maintain memory and context, and how they can be deployed as production-ready systems.
